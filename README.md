Aca la consigna de los ejercicios

Ejercicio 1-Vehículos
Desarrollar un sistema para administrar vehículos.
El sistema debe permitir:
● registrar vehículos
● calcular velocidad
● mostrar información
● realizar acciones especiales
Algunos vehículos pueden tener funcionalidades adicionales.






Ejercicio 2 - Empleados (en este me limité a hacerlo muy parecido al de vehiculos pq no me resultaba llamativo)
Desarrollar un sistema para empleados de una empresa.
El sistema debe:
● administrar empleados
● calcular salarios
● mostrar tareas
● generar reportes
No todos los empleados trabajan igual.





Ejercicio 3 - Videojuego
Desarrollar un sistema simple de personajes para un videojuego.
El sistema debe permitir:
● atacar
● defenderse
● usar habilidades
● mostrar estadísticas
No todos los personajes poseen las mismas habilidades.





Ejercicio 4 - Tienda Online
Desarrollar un sistema de compras online.
Debe permitir:
● administrar productos
● calcular precios finales
● aplicar descuentos
● realizar pagos
Existen distintos tipos de productos y pagos
